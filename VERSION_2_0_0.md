# Colorful Folders v2.0.0 - The Premium Update

Welcome to the most significant update to Colorful Folders! This version introduces five powerful new features to give you ultimate control over your file explorer aesthetics.

## What's New?

### 🌟 Active File Pathway Glow
Never lose track of your location again. The connecting lines and folder titles leading down to your *currently open file* now glow with increased brightness and a subtle shadow, creating a clear breadcrumb trail in your sidebar.

### 🎨 Custom Hex Color Palettes
You are no longer limited to our default themes. In the plugin settings, you can now enter a comma-separated list of your favorite Hex codes (e.g., `#eb6f92, #9ccfd8`). The plugin will automatically cycle through your personal brand colors.

### 🔮 Monochromatic Blending Mode
Want a cleaner, more uniform look? Switch the `Color Blending Mode` to **Monochromatic**. Every subfolder will inherit the specific color of its root parent, creating a neatly organized, color-coded structure for separate projects.

### 👻 Outline Only mode (Minimalist)
If you prefer a cleaner Obsidian interface, toggle **Outline Only**. This removes all background tints, keeping the file tree dark while preserving the vibrant, high-visibility connecting lines and colorful text titles.

### 🙈 Advanced Folder Exclusion
Clean up your sidebar by hiding the rainbow from utility folders. You can now define an `Exclusion List` in the settings (e.g., `templates, attachments, .obsidian`) to keep those folders standard and distraction-free.

---

## Technical Enhancements
- **Increased Guide Visibility**: Horizontal and vertical boundary lines are now matched at `2px` thickness and `0.8` opacity for better visual grouping.
- **Improved Spacing**: Cascading margins now ensure that deeply nested folders draw their vertical lines without overlapping or "stacking" into a mess.
- **Smart Defaults**: Version 2.0.0 ships with **Solid Vivid Color** as the default root style for an immediate "wow" factor upon installation.

Thank you for using Colorful Folders! Stay colorful. 🌈
