# Colorful Folders v3.0.0 - The Pro Feature Update 🌈

Version 3.0.0 is a massive leap forward, transforming Colorful Folders from a visual utility into a powerful vault management tool with premium aesthetics and smarter organization.

## 🚀 What's New in v3.0.0?

### 🖱️ Context Menu Color Overrides
You now have total control. Right-click any folder in your file explorer and select **"Set Custom Folder Color"** to assign a specific Hex code. This color will persist regardless of your global palette or cycling settings.

### 🔥 Activity Heatmap Mode
Switch your `Color Generation Mode` to **Activity Heatmap**. Your folders will now visually "glow" based on their contents:
- **Bright Red/Pink**: Modified within the last 24 hours.
- **Orange/Yellow**: Modified within 3 days.
- **Green**: Modified within a week.
- **Blue**: Modified within 15 days.
- **Cool Lavender**: Stale folders (over 30 days).

### 🏷️ Auto-Generate Emoji Icons
Stop manually adding emojis. Enable **Auto-Icons** to automatically inject contextual emojis based on your folder names:
- `Journal` -> 📅 | `Images` -> 🖼️ | `Project` -> 🚀 | `Archive` -> 📦 | `Settings` -> ⚙️

### 🧊 Glassmorphism & Focus Mode
- **Glassmorphism**: Adds a high-end backdrop blur effect to your folder backgrounds.
- **Focus Mode**: Automatically dims inactive top-level folders to 35% opacity when you are deep-working in a specific path, reducing visual noise.

### 🌬️ Breathing Active Path
The Active File Pathway now features an optional **"Breathing" animation**, causing the glowing connecting lines to pulse softly as you navigate.

### 🏳️‍🌈 Rainbow Root Text
Give your top-level structure a premium flair with **Rainbow Text Gradients**. You can choose to float this text freely or keep it over your solid background blocks.

---

## 🛠️ Internal Refinements
- **Native UI Components**: The custom color picker now uses Obsidian's native APIs for better theme compatibility.
- **Reorganized Settings**: A completely rebuilt, categorized settings menu for easier navigation.
- **Optimized Performance**: Heatmap calculations are efficient and updated on vault load.

Thank you for being part of the Colorful Folders journey. Version 3.0.0 is our most ambitious release yet!
