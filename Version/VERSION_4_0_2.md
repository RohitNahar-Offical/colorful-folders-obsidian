# Colorful Folders 4.0.2 - Optimized Factory Defaults 🎨

Version 4.0.2 hard-codes the user's preferred global configuration as the new factory defaults for the plugin, while keeping personal data private.

## ✨ What's New in 4.0.2?

### 🛠️ New Global Defaults
- **Global Settings Sync**: Your preferred settings for palette (Pastel Dreams), opacity, and animations are now the default baseline for all new installations.
- **Privacy First**: Your custom folder styles and personal presets were explicitly **excluded** from the source code, keeping your vault's structure private.
- **Clean Reset**: Resetting the plugin to defaults will now return it to these optimized global settings.
