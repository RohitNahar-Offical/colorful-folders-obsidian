# Colorful Folders 4.0.6 - Custom Icon Packs & UI Overhaul 🚀

Version 4.0.6 introduces a powerful custom icon management system and a completely redesigned, more compact styling interface.

## ✨ What's New in 4.0.6?

### 📦 Custom Icon Pack System
- **Your Icons, Your Way**: Added a dedicated 'Icon Packs' management tab in the settings. You can now add your own SVG icons by ID.
- **Bulk Internet Import**: Import entire icon sets from the internet by providing a URL to a JSON icon pack.
- **Manual SVG Addition**: Directly paste SVG code to create custom icons for your folders and files.
- **Universal Picker Integration**: All custom icons are automatically registered and appear in the plugin's icon picker alongside standard Lucide icons.

### 🍱 Redesigned Compact Modal
- **Horizontal Picker Layout**: Changed the visual color picker to a horizontal flow, drastically reducing vertical space and scrolling.
- **Card-Based UI**: Settings are now logically grouped into distinct sections (Background, Text, Icon) with subtle borders for a more professional, organized feel.
- **Surgical Apply Buttons**: Moved "Apply" and "Reset" buttons directly into the section headers for better accessibility.
- **Dedicated Icon Management**: Moved Icon Color and Selection into its own tab within the modal to reduce clutter in the Appearance tab.

### 🛠️ Stability & Performance
- **Lifecycle Fixes**: Resolved critical syntax errors and method scoping issues to ensure stable plugin loading.
- **Custom Icon Registration**: Efficient startup logic registers all custom icons with Obsidian on plugin load.
- **Settings Optimization**: Added a new tab-switching logic for a smoother configuration experience.

Thank you for your fantastic feedback — 4.0.6 makes Colorful Folders even more extensible and professional!
