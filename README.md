# Colorful Folders

Automatically adds beautiful premium preset colors to all folders and subfolders in Obsidian. 

## Features
- Choose between Vibrant Rainbow & Muted Dark Mode color palettes.
- Set root folder appearance to either Solid Vivid Color or a Translucent Glow.
- Customizable opacity settings for root folders, subfolders, and opened folder backgrounds.
- Instantly updates colors when toggled or modified in settings.

## Installation

### From Obsidian Community Plugins
*(Pending Approval)* Once approved, you will be able to search "Colorful Folders" in the Obsidian community plugins tab and install directly.

### Manual Installation
1. Download the latest release from the GitHub releases page.
2. Extract the contents (`main.js` and `manifest.json`) into your vault's `.obsidian/plugins/colorful-folders/` folder.
3. In Obsidian settings, go to Community plugins, disable Safe Mode, and enable Colorful Folders.
