# Colorful Folders v3.5.0 - The "Pro" Customization Update 🌈

Version 3.5.0 is a professional-grade enhancement that integrates individual file styling, advanced typography, and smart inheritance to give you granular control over your vault's visual organization.

## 🚀 What's New in v3.5.0?

### 📁 Individual File Styling
The power of Colorful Folders is no longer limited to folders. You can now right-click any **File** and select **"Set Custom File Style"**. 

### 🅱️ Advanced Typography
Take full control of your folder and file titles with new typography controls:
- **Bold & Italic Toggles**: Make key projects or files stand out instantly.
- **Custom Text Color**: Override the default theme colors for specific items.
- **Improved Opacity**: Highly refined opacity controls that dim both background and text for a premium, consistent look.

### 🔄 Smart Inheritance
Manage large vaults with ease using our new inheritance engine:
- **Apply to Subfolders**: Force a specific style down the entire folder tree.
- **Apply to Files**: Instantly style all files within a folder to match its aesthetic.

### 📋 Simple Preset System
Create your own "Design Language". Save any combination of color, typography, and opacity as a **Preset** in the style modal and apply it to other items with one click.

---

## 🛠️ Internal Refinements & Fixes
- **Refactored CSS Engine**: A specialized `traverse` engine that optimizes CSS generation and reduces duplicate rules.
- **Opacity Parity**: Fixed the mismatch where custom opacity only affected the background; it now correctly dims the entire navigation item.
- **Contrast Logic**: Improved auto-contrast for custom colors to ensure text is always readable (defaults to high-contrast white).
- **Classic Defaults**: Preserved the "Always Bold" look for folders to ensure version 3.5.0 feels familiar while adding new power.

Thank you for using Colorful Folders. Version 3.5.0 makes your Obsidian vault feel like a premium, professional workspace!
